# FinalProJectYear2-1

Project Name: Smart Persona (FYNEX) Type: Final Project (Year 2, Semester 1)

Project Overview: Smart Persona is a next-generation job matching platform designed to bridge the gap between talent and opportunity using advanced Artificial Intelligence. Going beyond traditional resume scanning, the platform leverages AI to analyze a candidate's unique "Persona" and personality traits. This creates a digital "AI Twin" that ensures a holistic match, aligning technical skills with company culture for the perfect fit.

Key Features:

AI-Driven Matching Engine: Utilizes multi-provider AI models (Gemini, OpenAI, Groq, Mistral) to deliver highly accurate and personalized job recommendations.
Intelligent Resume Parsing: Automatically extracts and structures data from uploaded resumes (PDFs/Images), streamlining the onboarding process.
Personality & Culture Fit: Analyzes user personality to provide insights that go beyond the resume, ensuring compatibility with potential employers.
Immersive User Experience: Features a cutting-edge, high-performance UI with fluid animations and a premium design aesthetic to engage users.
